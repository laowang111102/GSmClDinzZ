# 停车场管理系统 java652

## 项目概述

停车场管理系统是基于Springboot和Vue技术栈开发的一套软件系统，旨在提供停车场的高效管理解决方案。

## 技术栈

- 后端：Springboot
- 前端：Vue

## 功能模块

系统主要包括以下功能模块：

- 顾客端
- 管理员后台界面

## 系统角色

- 顾客
- 管理员

## 运行环境

系统支持各类操作系统，具体环境需求如下：

- JDK 1.8 或更高版本
- MySQL 5.7 或更高版本

## 部署步骤

1. 准备开发环境
2. 导入数据库文件
3. 部署后端服务
4. 部署前端服务
